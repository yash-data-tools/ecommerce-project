# Ecommerce Project

A full-stack ecommerce web application built with React, Vite, Node.js, and Express.

## Live Demo

[View Real Frontend Website](https://yks-ecommerce-project.vercel.app)

## Features

- Browse products
- Search products
- Add products to cart
- Update cart quantity
- Remove items from cart
- Checkout page
- Place orders
- View previous orders
- Track order status
- Responsive user interface

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Axios

### Backend
- Node.js
- Express.js

## Backend Repository

[View Backend](https://github.com/yash-data-tools/ecommerce-backend)

## Installation

```bash
npm install
npm run dev
```
